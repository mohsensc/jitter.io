# RULES

Mouse will be moved when:
-> Not moving
-> Every 6 and 7 seconds
-> Mouse movements are slow

Have fun!

## Instalation

```bash
pip install -r requirements.txt
```
If doesn't work, check requirements.txt for manual installation guide

## Disclaimer

App needs permission to control mouse and receive inputs from device. Some text editors (e.g. VS Code) also need additional perms to allow these functions. There is no malicious code in this program, but allow access at own risk and disable access afterwards.
